# UI Review Rubric

Prioritize in this order:

1. Broken layout: overlap, clipping, off-screen content.
2. Readability: low contrast, tiny text, excessive line length.
3. Navigation: inconsistent navbar positions, unclear active state.
4. Responsiveness: mobile wrapping, button overflow, grid collapse.
5. Visual hierarchy: primary action not clear, headings too large in compact panels.
6. Polish: spacing rhythm, border consistency, hover states.

Start with issues a real user would notice or be blocked by.
