# Python Error Patterns

- `ModuleNotFoundError`: package missing or wrong virtual environment.
- `ImportError`: package exists but exported name or version may differ.
- `SyntaxError`: source cannot parse; fix the line shown first.
- `IndentationError`: whitespace block issue.
- `KeyError` for env/config: missing config value.
- `Address already in use`: port conflict.
- `PermissionError`: filesystem or executable permission problem.
